# Canvas LMS

Canvas is the learning management system (LMS) used across SCPS for course management, assignments, grading, communication, and more. It's the central hub where students access their coursework and where you manage your classroom digitally.

!!! info "SCPS-Specific Notes"
    - Canvas courses are automatically created from your **SIS roster**. You do not need to create courses manually.
    - Your Canvas account links to **MasteryConnect** for standards-based assessment data. See the [MasteryConnect guides](../masteryconnect/index.md) for details.
    - Students access Canvas at your division's Canvas URL. The **Canvas Student app** is also available on Chromebooks and mobile devices.
    - Parent/guardian access is available through the **Canvas Parent app** — encourage families to set this up.
