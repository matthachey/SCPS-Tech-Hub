# Canvas Tips and Best Practices

A collection of power-user strategies, common mistakes to avoid, and SCPS-specific tips for getting the most out of Canvas.

**Time to complete:** ~5 minutes

---

## Course design principles

!!! tip "Key principle"
    Design your Canvas course from the **student's perspective**. If a student opens your course and can't figure out what to do in 10 seconds, simplify.

### The 3-click rule

Students should be able to find any current assignment within **3 clicks** of opening your course. If they have to dig through menus and scroll through long lists, they'll disengage.

### Less is more

- **Hide unused navigation items** (Settings → Navigation)
- **Only publish the current module** — keep future content hidden
- **Use consistent naming** — `Unit 3: Topic - Assignment Type` is scannable
- **Avoid uploading files when a Page works better** — Pages are mobile-friendly and accessible

---

## Top strategies

### 1. Use modules as your single source of truth

Put everything in modules in the order students should complete them. If an assignment lives in Modules, students always know where to look. Avoid scattering content across Assignments, Pages, and Files without a clear module structure.

### 2. Write self-contained assignment descriptions

Every assignment description should answer: What do I do? How do I do it? When is it due? What does success look like? If a student has to ask you a question that's answered in the description, the description needs to be clearer.

### 3. Use "Message Students Who" religiously

This Gradebook feature is the most underused tool in Canvas. Instead of guessing who's falling behind, let Canvas tell you. Message students who haven't submitted, scored below a threshold, or haven't been graded yet.

### 4. Copy courses between terms

Don't rebuild your course every semester.

1. Go to **Settings** → **Import Course Content**.
2. Select **"Copy a Canvas Course"**.
3. Choose the previous term's course.
4. Select what to import (all content, specific modules, etc.).
5. **Update your due dates** after import — this is the most common mistake.

### 5. Use Student View before publishing

Always check **Student View** (Settings → Student View) before making content live. You'll catch broken links, unpublished items, and confusing navigation.

## Common mistakes to avoid

!!! warning "Things to watch out for"
    - **Not publishing the course** — Students can't see anything until the course is published.
    - **Publishing everything at once** — Students get overwhelmed seeing 15 weeks of content. Publish one module at a time.
    - **Forgetting to update due dates after course copy** — Importing from last semester keeps the old dates. Update every one.
    - **Not setting a grading policy** — Check your assignment group weights before grades go home. One misconfigured weight can throw off every student's grade.
    - **Using Files instead of Pages** — Uploaded PDFs and docs are hard to read on Chromebooks and phones. Convert key content to Canvas Pages whenever possible.
    - **Ignoring the To Do list** — The To Do sidebar on your Dashboard shows ungraded submissions. Stay on top of it.

## SCPS-specific reminders

| Topic | Details |
|-------|---------|
| **Course creation** | Courses are auto-created from your SIS roster. Don't create courses manually. |
| **MasteryConnect integration** | Your Canvas courses link to MasteryConnect trackers. See [Linking MasteryConnect to Canvas](../masteryconnect/link-to-canvas.md). |
| **SIS grade sync** | Check with your admin about whether Canvas grades sync to your SIS (PowerSchool/other). This varies by building. |
| **Canvas Parent app** | Encourage families to download the Canvas Parent app for visibility into student grades and assignments. |
| **Canvas Student app** | Students can access Canvas on Chromebooks, phones, and tablets via the Canvas Student app. Test your assignments on mobile to ensure they work. |

## Quick reference

| I Want To... | Do This |
|--------------|---------|
| Create an assignment | Assignments → + Assignment |
| Build a quiz | Assignments → + Assignment → External Tool → New Quizzes |
| Organize content | Modules → + Module → add items |
| Grade submissions | Open assignment → SpeedGrader |
| See all grades | Grades (Gradebook) |
| Message the class | Announcements → + Announcement |
| Message one student | Inbox → Compose |
| Message students who didn't submit | Gradebook → column three dots → Message Students Who |
| Check student view | Settings → Student View |
| Copy last semester's course | Settings → Import Course Content |
| Publish the course | Home → Publish button |
