# Building a Quiz with New Quizzes

New Quizzes is Canvas's modern quiz engine with more question types, item banks, and auto-grading features. This guide covers creating a quiz from scratch and from item banks.

**Time to complete:** ~10 minutes

---

## Creating a New Quiz

1. Go to **Assignments** → click **"+ Assignment"**.
2. Set the **Submission Type** to **"External Tool"**.
3. Select **"New Quizzes"** from the tool list.
4. Set your **points**, **due date**, and **assignment group**.
5. Click **"Save"** or **"Save & Publish"**.
6. You'll be taken into the **New Quizzes editor** to build your questions.

Alternatively, go to **Quizzes** → click **"+ Quiz"** → select **"New Quizzes"**.

---

## Adding questions

In the New Quizzes editor, click **"+"** to add a question. Available question types:

### Auto-graded types (most efficient)

| Type | Description |
|------|-------------|
| **Multiple Choice** | One correct answer from a list |
| **Multiple Answer** | Select all that apply |
| **True/False** | Binary choice |
| **Matching** | Match items from two columns |
| **Ordering** | Arrange items in the correct sequence |
| **Fill in the Blank** | Students type a short answer (you define accepted answers) |
| **Hot Spot** | Students click on the correct area of an image |
| **Categorization** | Sort items into categories |

### Manually graded types

| Type | Description |
|------|-------------|
| **Essay** | Open-ended written response |
| **File Upload** | Students submit a file as their answer |
| **Numeric** | Students enter a number (with tolerance range) |

!!! tip "Mix your question types"
    A quiz with only multiple choice can be low-DOK. Mix in matching, ordering, and fill-in-the-blank to assess deeper understanding without needing manual grading.

## Using item banks

Item banks are reusable collections of questions. You can pull random questions from a bank so each student gets a different version.

### Creating an item bank

1. In the New Quizzes editor, click **"Item Banks"** in the left sidebar.
2. Click **"+ Bank"** and name it (e.g., `Unit 3 - American Revolution`).
3. Add questions to the bank by clicking **"+"** inside the bank.

### Adding bank questions to a quiz

1. In your quiz, click **"+"** to add a question.
2. Select **"Item Bank"** instead of creating a new question.
3. Choose the bank and how many questions to pull.
4. Canvas will randomly select questions from the bank for each student.

!!! info "Why use item banks?"
    Item banks let you create a large pool of questions and randomly select a subset for each student. This reduces cheating and lets you reuse questions across multiple quizzes without rebuilding them.

## Quiz settings

Before publishing, configure your quiz settings:

| Setting | Recommendation |
|---------|----------------|
| **Time Limit** | Set one for formal assessments; leave blank for practice quizzes |
| **Attempts** | 1 for summative; 2–3 for formative/practice |
| **Shuffle Questions** | Turn on to randomize question order per student |
| **Shuffle Answers** | Turn on to randomize answer choice order |
| **Show Results** | Choose when students see their results (after submission, after due date, or never) |
| **One Question at a Time** | Displays questions individually — good for preventing students from flipping ahead |

---

## Accommodations

Need to give a student extra time or additional attempts?

1. In the quiz settings, look for the **"Manage Student Access"** or **"Moderate"** option.
2. Find the student's name.
3. Add **extra time** (e.g., +15 minutes) or **extra attempts**.

This applies only to that student — the rest of the class keeps the default settings.

!!! tip "Set accommodations before students start"
    It's easier to set accommodations before the quiz opens. If a student has already started, you can still moderate, but it may require reopening their attempt.

---

## After the quiz

- **Auto-graded questions** will have scores available immediately.
- **Essay and file upload questions** need to be scored manually in SpeedGrader.
- View **quiz statistics** from the quiz page to see item analysis — which questions students missed most.
