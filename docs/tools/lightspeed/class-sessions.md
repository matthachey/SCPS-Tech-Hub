# Starting and Managing a Class Session

Lightspeed Classroom only monitors students during an **active class session**. This guide covers how to start, manage, and end sessions using the different schedule types.

**Time to complete:** ~5 minutes

---

## Understanding schedule types

Lightspeed offers three ways to run a class session:

| Schedule Type | What It Does | Best For |
|---------------|--------------|----------|
| **Bell Schedule** | Automatically starts/stops based on your school's bell times | Set by your district — no action needed from you |
| **Custom Schedule** | You set specific start and end times | Regular class periods you want to run the same time each day |
| **Ad Hoc** | You choose a duration and start it immediately | Study halls, flex periods, one-off sessions |

!!! info "At SCPS"
    Check with your ITRT about whether bell schedules have been configured for your building. If not, you'll use custom or ad hoc schedules.

---

## Option A: Using a custom schedule

Custom schedules let you set a recurring time for your class session.

1. Click on a class from your **My Classes** homepage.
2. Click **Settings** (gear icon) at the bottom left.
3. Select **"Custom"** schedule.
4. Set the **start time** and **end time** for your class.
5. Click **"Save Settings"**.

The session will automatically start and stop at those times each school day.

!!! warning "One custom schedule per class per day"
    You can only set one custom schedule per class. If you teach the same group at two different times, use the custom schedule for one and an ad hoc session for the other.

---

## Option B: Using an ad hoc session

Ad hoc sessions are perfect for flexible schedules or one-time monitoring needs.

1. Click on a class from your **My Classes** homepage.
2. Click the **"Start Class"** button (or look for the ad hoc schedule option).
3. Select a **duration** (e.g., 30 minutes, 45 minutes, 1 hour).
4. Click **"Start"** — monitoring begins immediately.

The session will automatically end when the timer runs out, or you can stop it manually.

---

## Ending a session early

If class ends early or you need to stop monitoring:

1. Open the class you want to stop.
2. Click **"End Class"** or stop the ad hoc timer.
3. Student devices are released from monitoring.

!!! tip "Don't forget to end your session"
    If your session is still running when the next teacher tries to monitor the same students, those students will show up as "Available to Add" in the other teacher's class. End your session promptly to avoid this.

---

## Troubleshooting

??? question "My custom schedule isn't starting automatically"
    - Go to **Settings** → **Custom** and verify the times are correct
    - Make sure you clicked **"Save Settings"**
    - Try refreshing the page
    - If the issue persists, contact your ITRT — there are occasional known issues with custom schedules

??? question "Students aren't showing up in my class"
    - Make sure the students have **joined your Google Classroom** (Lightspeed syncs from Google Classroom rosters)
    - Verify the student's device is **online and on campus**
    - Check if the student is in **"Available to Add"** — another teacher's session may still be active
