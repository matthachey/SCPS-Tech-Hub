# Pushing Links to Students

Instead of writing a URL on the board or telling students to "go to this website," Lightspeed lets you push a link directly to student browsers. One click from you, and every student is on the same page — literally.

**Time to complete:** ~3 minutes

---

## Push a link to the whole class

1. From your class view, click the **"Push Link"** button in the top toolbar.
2. **Paste or type the URL** you want students to visit.
3. Select **"Entire Class"** (or choose specific students/groups).
4. Click **"Send"**.

The link opens automatically in a new tab on every student's device.

---

## Push a link to one student or a group

1. **For one student**: Click on their thumbnail, then click the **Push Link** icon.
2. **For a group**: Select the group from the left sidebar, then click **Push Link**.
3. Enter the URL and send.

---

## When to use Push Link

| Scenario | Why Push Link Helps |
|----------|---------------------|
| Starting a new activity | Gets everyone to the same page instantly — no time wasted typing URLs |
| Sharing a Google Doc or Form | No need to post a link in Google Classroom first |
| Redirecting a student | Push the correct resource to a student who's on the wrong site |
| Differentiated resources | Push different links to different groups |

!!! tip "Pair with Allow Mode"
    Push the link first, then turn on **Allow Mode** so students can only access that site (and any others you've allowed). This keeps them focused on exactly what you want.
