# Setting Up Categories and Calculations

Grade categories and calculation settings determine how individual assignment scores roll up into quarter, semester, and final grades. Getting this right is critical.

**Time to complete:** ~10 min

---

## Understanding categories

Categories group similar assignments together (e.g., Classwork, Homework, Assessments, Projects). Each category can be weighted differently in the final grade calculation.

!!! warning "Use district categories"
    SCPS may provide **pre-set district categories**. If so, **do not create your own** — grades assigned to custom categories that aren't recognized by the district calculation setup won't factor into final grades. Check with your ITRT or building admin before making changes.

---

## Viewing and adjusting your categories

1. Click **Grading** → **Categories**.
2. You'll see a list of available categories.
3. To edit a category, click the **pencil icon** next to it.
4. To hide categories you don't use, click the pencil and set it to **Inactive**. This cleans up your dropdown list when creating assignments.

### Setting category defaults

For each category you use, you can set defaults that auto-apply when you create new assignments:

1. Click the pencil icon on a category.
2. Set default **score type** (points, percentage, letter grade).
3. Set default **points possible**.
4. Set **publishing options** for the parent/student portal.
5. Click **Save**.

---

## Setting up Traditional Grade Calculations

This is where you tell PowerTeacher Pro how to weight your categories.

1. Click your **name** (bottom left) → **Settings**.
2. Select **"Traditional Grade Calculations"**.
3. You'll see folders for each reporting term (Q1, Q2, S1, F1, etc.) — these are set by the district.
4. Click the **pencil icon** under Actions for a term.
5. Choose **"Category Weighting"** as the calculation type.
6. Set the **percentage weight** for each category.

| Category | Example Weight |
|----------|----------------|
| Classwork | 30% |
| Homework | 20% |
| Assessments | 40% |
| Projects | 10% |

!!! warning "Weights must match your school's grading policy"
    Check with your building administration for the correct weighting scheme. This may be standardized across your school or grade level.

---

## Copying calculations to other terms and classes

Once you've set up one quarter correctly, you don't need to redo it:

### Copy within a class (Q1 to Q2, Q3, Q4)

1. In the Traditional Grade Calculations screen, click the **gear icon** (top right).
2. Select **"Within a Class"**.
3. Choose the source term and destination term(s).
4. Check **"Recalculate final grades"**.
5. Click **Validate**, then **Copy**.

### Copy to other classes

1. Same gear icon → select **"To Other Classes"**.
2. Choose the source class and destination class(es).
3. Validate and Copy.
