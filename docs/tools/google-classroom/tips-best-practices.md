# Google Classroom Tips and Best Practices

A collection of power-user strategies, common mistakes to avoid, and SCPS-specific tips.

**Time to complete:** ~5 min

---

## Top strategies

### 1. Use topics religiously

A Classwork page without topics is a disaster for students. Create topics before you start posting, and assign every item to a topic. Keep the current topic at the top by reordering.

### 2. "Make a copy for each student" is your best friend

For worksheets, templates, and graphic organizers, always select **"Make a copy for each student"** when attaching a Google Doc. Each student gets their own copy, it's automatically named with their name, and when they turn it in, you can grade it individually.

### 3. Schedule assignments in advance

Build your week's assignments in one sitting and schedule them to post on the right days. This saves time and ensures students get a consistent flow of work.

### 4. Use "Grade All" for completion-based work

Exit tickets, bell ringers, classwork checks — use the **"Grade All"** feature in the Grades tab to give everyone the same score at once. Paired with "Automatically return," it takes seconds.

### 5. Check "Turned in" counts before class ends

Before students leave, glance at the Classwork page to see how many have turned in the assignment. Call out the count: "I see 18 out of 24 turned in — if you haven't submitted, do it now." This catches stragglers in real time.

### 6. Copy your class for new semesters

Don't rebuild from scratch every term:

1. On the Google Classroom homepage, click **"+"** → **"Create class"**.
2. Or go to an existing class → **gear icon** → find the option to **copy** the class.
3. All assignments, materials, and topics carry over.
4. **Update due dates** — this is the one thing that doesn't transfer cleanly.

## Common mistakes to avoid

!!! warning "Watch out for these"
    - **Not using topics**: Students see one long, unsorted list. They'll miss things.
    - **Choosing "Students can edit file" when you meant "Make a copy"**: One student edits the doc and everyone's copy changes. Always double-check the sharing option.
    - **Forgetting to return work**: Grades are invisible until you return them. Students think they haven't been graded.
    - **Posting to the Stream instead of Classwork**: The Stream is a feed — things scroll away. Assignments and materials belong on the Classwork page.
    - **Not setting up grading categories before posting assignments**: If you add categories later, you'll need to go back and assign categories to every existing assignment.
    - **Leaving the class code open all year**: After all students have joined, disable or reset the class code to prevent unauthorized joins.

## Google Classroom vs. Canvas at SCPS

Since SCPS uses both platforms, here's when to use which:

| Use Google Classroom For | Use Canvas For |
|--------------------------|----------------|
| Quick assignments using Google Docs/Sheets/Slides | Formal course structure with modules |
| Daily classwork and bell ringers | Summative assessments and New Quizzes |
| Simple grading with "Grade All" | Weighted gradebook with detailed reporting |
| Classes that live entirely in the Google ecosystem | MasteryConnect integration and standards-based tracking |
| Lightweight communication via Stream | SpeedGrader annotation and detailed rubric feedback |

!!! info "Know your primary gradebook"
    Check with your building administration about which platform is the **gradebook of record** — the one where official grades live. Don't split grades between two systems without a clear plan.

## Quick reference

| I Want To... | Do This |
|--------------|---------|
| Create an assignment | Classwork → + Create → Assignment |
| Create a quiz | Classwork → + Create → Quiz assignment |
| Share a resource | Classwork → + Create → Material |
| Ask a quick question | Classwork → + Create → Question |
| Make an announcement | Stream → "Announce something" |
| Grade work | Click assignment → student work page → enter grade |
| Grade all at once | Grades tab → three-dot menu → "Grade all" |
| Return work | Student work page → check students → Return |
| Export grades | Grades → Settings → Copy all grades to Google Sheets |
| Invite guardians | People → three dots on student → Invite guardian |
