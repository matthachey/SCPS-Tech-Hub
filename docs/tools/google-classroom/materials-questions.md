# Posting Materials and Questions

Not everything in Classroom needs to be a graded assignment. **Materials** let you share resources, and **Questions** let you gather quick student responses.

**Time to complete:** ~5 min

---

## Posting materials

Materials are ungraded resources — readings, reference documents, videos, links.

1. Go to **Classwork** → **"+ Create"** → **"Material"**.
2. Add a **title** and optional **description**.
3. **Attach** files, links, Drive documents, or YouTube videos.
4. Select a **topic** to keep things organized.
5. Click **"Post"**.

Materials don't appear in the Grades page and don't have due dates. They're purely reference content.

### When to use materials vs. assignments

| Use a Material | Use an Assignment |
|----------------|-------------------|
| Sharing a study guide | Completing a worksheet |
| Posting a video to watch | Responding to a video with questions |
| Sharing a link to a resource | Submitting work based on the resource |

---

## Posting questions

Questions are lightweight, graded or ungraded prompts that collect a quick student response.

1. Go to **Classwork** → **"+ Create"** → **"Question"**.
2. Choose the question type:
    - **Short answer** — Students type a response
    - **Multiple choice** — Students select from options you provide
3. Add the question text.
4. Set **points** (or ungraded), **due date**, and **topic**.
5. Options:
    - **Students can reply to each other** — turns it into a mini-discussion
    - **Students can edit their answer** — allows revision
6. Click **"Ask"**.

### When to use questions

| Scenario | Question Type |
|----------|---------------|
| Exit ticket | Short answer: "What's one thing you learned today?" |
| Quick poll | Multiple choice: "Which topic should we review tomorrow?" |
| Bell ringer | Short answer: "Define [vocabulary term] in your own words." |
| Check for understanding | Multiple choice with a content question |

!!! tip "Questions are great for daily engagement"
    A quick question at the start or end of class takes 2 minutes to create and gives you immediate insight into student thinking — without the overhead of a full assignment.
