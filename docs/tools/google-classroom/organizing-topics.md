# Organizing Classwork with Topics

Topics are folders that group your assignments, quizzes, and materials on the Classwork page. Without topics, everything piles up in one long list.

**Time to complete:** ~5 min

---

## Creating a topic

1. Go to the **Classwork** tab.
2. Click **"+ Create"** → **"Topic"**.
3. Name the topic (e.g., `Unit 3: The American Revolution` or `Week 5`).

---

## Choosing a topic structure

| Approach | Example Topics | Best For |
|----------|---------------|----------|
| **By unit** | `Unit 1: Foundations`, `Unit 2: Growth`, `Unit 3: Conflict` | Content-driven courses with clear units |
| **By week** | `Week 1`, `Week 2`, `Week 3` | Courses with consistent weekly cadence |
| **By type** | `Classwork`, `Homework`, `Assessments`, `Resources` | When you want work grouped by category |
| **Hybrid** | `Unit 3 - Classwork`, `Unit 3 - Assessments` | Best of both worlds |

!!! tip "Most K-12 teachers prefer by unit or by week"
    Students find it easiest when the Classwork page mirrors how class time is structured. If you teach in units, use unit topics. If you follow a weekly rhythm, use week topics.

---

## Assigning a topic to classwork

When creating any assignment, quiz, or material:

1. In the creation form, find the **"Topic"** dropdown.
2. Select an existing topic or create a new one right there.

You can also **drag and drop** existing assignments between topics on the Classwork page.

---

## Reordering topics and items

- **Drag topics** up or down to reorder them. Put the current topic at the top.
- **Drag items** within or between topics to reorganize.
