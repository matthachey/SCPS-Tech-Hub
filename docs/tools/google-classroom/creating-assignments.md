# Creating Assignments

Assignments are the primary way students receive and submit work in Google Classroom. This guide covers creating assignments with attachments, due dates, and submission options.

**Time to complete:** ~8 min

---

## Creating a basic assignment

1. Go to the **Classwork** tab.
2. Click **"+ Create"** → **"Assignment"**.
3. Fill in the details:

| Field | What to Enter |
|-------|---------------|
| **Title** | Clear, descriptive name (e.g., `Chapter 5 Reading Response`) |
| **Instructions** | What students need to do — be specific |
| **Points** | Point value (or select "Ungraded") |
| **Due date** | When the assignment is due (date and optional time) |
| **Topic** | Which topic/category this belongs to (see [Organizing with topics](organizing-topics.md)) |
| **Grade category** | If using weighted grading, select the category |

---

## Attaching files and resources

Click **"Add"** to attach materials:

| Attachment Type | How It Works |
|-----------------|--------------|
| **Google Drive** | Attach Docs, Sheets, Slides, or other Drive files |
| **Link** | Attach a URL to a website or resource |
| **File** | Upload a file from your computer |
| **YouTube** | Embed a YouTube video (if enabled by your admin) |

### The three sharing options for Google Drive attachments

When you attach a Google Doc, Sheet, or Slide, you choose how students interact with it:

| Option | What Happens | Best For |
|--------|--------------|----------|
| **Students can view file** | Read-only — students can see it but not edit | Reference documents, readings, rubrics |
| **Students can edit file** | Everyone shares one copy and can edit it | Collaborative documents, shared brainstorms |
| **Make a copy for each student** | Each student gets their own copy with their name on it | Worksheets, templates, individual assignments |

!!! tip "Make a copy for each student"
    This is the option you'll use most often. It creates a personal copy for every student, automatically named with their name. When they turn it in, you can grade each student's individual file.

---

## Creating files directly in the assignment

Click **"Create"** to make a new Google Doc, Sheet, Slide, Drawing, or Form right from the assignment. The new file is automatically attached.

This is great for creating a worksheet template on the fly — create it, and it's ready for students to use.

---

## Assigning to specific students

By default, assignments go to **all students**. To assign to specific students:

1. Under **"For"**, click the dropdown showing "All students."
2. Deselect "All students."
3. Search for and select the specific students.

This is useful for differentiated assignments, makeups, or accommodations.

---

## Scheduling assignments

You don't have to post immediately:

1. Instead of clicking **"Assign"**, click the dropdown arrow next to it.
2. Select **"Schedule"**.
3. Pick a **date and time** for the assignment to post automatically.

!!! tip "Plan your week in advance"
    Create all your assignments on Monday morning (or Sunday night) and schedule them to post throughout the week. Students get a steady drip of work without you having to be at your computer at specific times.

---

## Saving drafts

If you're not ready to post:

1. Click the dropdown arrow next to "Assign."
2. Select **"Save draft"**.
3. The assignment saves but is not visible to students.
4. Return to it later from the Classwork tab to finish and post.

---

## Posting to multiple classes

1. Under **"For"**, click the dropdown.
2. Select additional classes.
3. The same assignment posts to all selected classes.

!!! warning "Multiple classes share the same assignment"
    If you edit the assignment later, the change applies to all classes it was posted to. If you need different versions for different classes, create separate assignments.
