# Contributing

Want to suggest a new guide, report an error, or contribute content? Here's how.

## Suggest a new guide

If there's a tool or task you'd like a guide for, reach out to your building ITRT with:

- The **tool name** and **specific task** you need help with
- Any **screenshots** of where you're getting stuck
- Whether this is something your **whole team** could benefit from

## Report an error

If you find incorrect information in a guide:

- Note the **guide title** and **step number** where the error is
- Describe what's **wrong** and what the **correct** information should be
- Send it to your building ITRT

## Contribute a guide

If you're an ITRT or tech-savvy teacher who wants to write a guide:

1. Guides are written in **Markdown** — a simple text formatting language
2. Follow the existing guide templates for structure and tone
3. Submit your draft to the Tech Hub maintainer for review

!!! info "Guide style tips"
    - Write for a teacher who has **never used the tool before**
    - Use **numbered steps** for procedures
    - Include **tips, warnings, and notes** using callout boxes
    - Add **screenshot placeholders** where visuals would help
    - End every guide with a **"What's next?"** section linking to related guides
