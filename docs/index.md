## Our Tools

| Tool | What It's For |
|------|---------------|
| **MasteryConnect** | Standards-based assessment, tracking mastery, data-driven instruction |
| **Canvas LMS** | Course management, assignments, grading, communication |
| **Google Classroom** | Assignments, announcements, classroom workflow |
| **Gemini for Workspace** | AI-powered assistance in Google Docs, Sheets, Slides, and Gmail |
| **Lightspeed Classroom** | Real-time classroom device management and monitoring |
| **Diffit** | AI-generated differentiated reading materials and activities |
| **PowerTeacher Pro** | Official gradebook (PowerSchool), GradeSync from Canvas and Classroom, report cards |
